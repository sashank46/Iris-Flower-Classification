# Iris Flower Classification

## Objective

The objective of this project is to classify iris flowers into one of three species:

* Setosa
* Versicolor
* Virginica

using machine learning classification algorithms based on flower measurements.

## Dataset

* Built-in Iris Dataset from Scikit-learn
* Number of samples: 150
* Number of features: 4

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Algorithm

* Logistic Regression

## Steps Performed

1. Loaded the Iris dataset.
2. Explored the dataset using summary statistics and scatter plots.
3. Split the dataset into training and testing sets.
4. Trained a Logistic Regression model.
5. Predicted the flower species.
6. Evaluated the model using:

   * Accuracy
   * Classification Report
   * Confusion Matrix

## Output

The model predicts the species of iris flowers with high accuracy.
